# Elastichsearch Kibana 8

Example virtualization of Elasticsearch and Kibana 8.

## ✨Features

Kibana 8 connected to a single node Elasticsearch 8.

## 🚀Getting Started

### 🐳Docker

Start the containers with the docker-compose: `docker-compose up`
